# ⚡ Nucleon CLI

Developer Workflow Engine - Project Inspector + Git Helper

## Installation

```bash
npm install -g nucleon-cli
```

## Commands

### `nucleon init`
Bootstrap a new project with templates for Node.js, Next.js, Python, or CLI tools.

### `nucleon doctor`
Check your development environment for required tools.

### `nucleon analyze`
Analyze your project structure, detect large files, and get suggestions.

### `nucleon commit`
Generate smart commit messages based on staged files.

## Development

```bash
npm install
npm run build
npm start
```
